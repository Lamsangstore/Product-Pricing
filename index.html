2 ต้นทุนต่อชิ้น (บาท) (ใช้ร่วมทุกแพลตฟอร์ม)

3 ค่าแต่ละแพลตฟอร์ม (แยกกันแต่ละที่)

💡 เช็กกำไรจริง —

⚖️ เทียบราคาข้ามแพลตฟอร์ม

แก้ค่าการตลาด % / ค่าธรรมเนียม % / ค่าชำระเงิน % / ค่าส่ง / ราคาขายจริง ของแต่ละแพลตฟอร์มได้ตรงในตาราง (แยกกันแต่ละที่ ระบบจำค่าล่าสุดให้) — “ราคาที่ควรตั้ง” คือราคาที่จะได้กำไรตามเป้า ส่วน “กำไรจริง” คิดจากราคาขายจริงที่กรอก

แพลตฟอร์ม ค่าการตลาด % ค่าธรรมเนียม % ค่าชำระเงิน % ค่าส่ง (บาท) ราคาที่ควรตั้ง ราคาเผื่อ ราคาขายจริง กำไรจริง/ชิ้น คุ้มทุน/เดือน

📈 กราฟจุดคุ้มทุน — ยิ่งขายยิ่งกำไร

พื้นที่แดง = ยังขาดทุน • เส้นตัดศูนย์ = จุดคุ้มทุน • พื้นที่เขียว = เริ่มมีกำไร • จุดน้ำเงิน = เป้ายอดขายที่ตั้งไว้

📁 รายการสินค้าที่บันทึก

แต่ละรายการเก็บค่าครบทุกแพลตฟอร์ม — กดไอคอน ✎ เพื่อโหลดข้อมูลทั้งหมดกลับมา

สินค้า / แพลตฟอร์ม ค่าธรรมเนียม ราคาที่ควรตั้ง ราคาเผื่อ ราคาขายจริง กำไรจริง/ชิ้น คุ้มทุน/เดือน
ยังไม่มีสินค้าที่บันทึก — กรอกข้อมูลแล้วกด “บันทึกสินค้านี้”
สูตรการคำนวณ & ต้นทุนอื่นที่ควรพิจารณาเพิ่ม
สูตรตั้งราคา: ราคาขาย = (ต้นทุนบาท + กำไรที่ต้องการ − เครดิต VAT ซื้อ) ÷ (1 − %การตลาด − %แพลตฟอร์ม − %ชำระเงิน − %VAT)
เพราะค่าการตลาด/ค่าธรรมเนียม/VAT คิดเป็น % ของราคาขาย ไม่ใช่ของต้นทุน จึงต้องหารกลับ ไม่ใช่บวกเพิ่มจากต้นทุน

จุดคุ้มทุน: จำนวนชิ้น/เดือน = ต้นทุนคงที่ต่อเดือน ÷ กำไรต่อชิ้น
